--- layout: post title: Operator module in Python tags: - Lisp - Python status: publish type: post published: true meta: {} --- Python has a fantastic module called 'operator'. The full documentation can be found in: http://docs.python.org/library/operator It has three main objects: attrgetter : After, f=attrgetter('name'), the call f(r) returns r.name. Looks for attribute itemgetter : After, f=itemgetter(2), the call f(r) returns r[2]. Looks for indexed item methodcaller  : After, f=methodcaller('name'), the call f(r) returns r.name(). Looks for method [sourcecode language="python" wraplines="false" collapse="false"] >>> import operator >>> l_of_l = [['a', 5, 'b'], ['z', 1, 'y'], ['p', 4, 'g']] >>> sorted(l_of_l, key=operator.itemgetter(1)) [['z', 1, 'y'], ['p', 4, 'g'], ['a', 5, 'b']] >>> sorted(l_of_l, key=operator.itemgetter(0)) [['a', 5, 'b'], ['p', 4, 'g'], ['z', 1, 'y']] >>> sorted(l_of_l, key=operator.itemgetter(2)) [['a', 5, 'b'], ['p', 4, 'g'], ['z', 1, 'y']] [/sourcecode] Anything which can compete with the first example above is the following Lisp code
(setf l_of_l '((z 1 y) (p 4 g) (a 5 b)))
(sort l_of_l #'(lambda (a b) (< (car (cdr a))
                                (car (cdr b)))))
